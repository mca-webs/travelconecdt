# Travel Connect Login

Tela de login moderna com mapa animado em canvas, desenvolvida com React, TypeScript, Tailwind CSS e Framer Motion.

## Funcionalidades

- Mapa mundi animado com rotas de viagem
- Login com Google (UI)
- Formulário de email/senha com validação
- Animações suaves com Framer Motion
- Design responsivo (mobile/desktop)
- Tema claro moderno
